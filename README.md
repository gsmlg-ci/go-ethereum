# go-ethereum

[![Build](https://github.com/gsmlg-ci/go-ethereum/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/go-ethereum/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/go-ethereum`
- `ghcr.io/gsmlg-dev/go-ethereum`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/go-ethereum:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/go-ethereum:latest
```

## Build

```bash
docker build -t gsmlg/go-ethereum .
```
